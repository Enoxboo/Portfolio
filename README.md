# Portfolio - Matteo Marquant

Portfolio personnel présentant mes compétences en développement web, mes projets et mon parcours en informatique.

## Technologies utilisées

- **React 19** - Framework JavaScript pour l'interface utilisateur
- **Tailwind CSS 4** - Framework CSS utility-first pour le design
- **Vite** - Build tool moderne et performant
- **JavaScript ES6+** - Langage de programmation

## Fonctionnalités

### Interface
- Design responsive adapté à tous les écrans (mobile, tablette, desktop)
- Animations fluides et interactions soignées
- Arrière-plan interactif avec effet nebula
- Navigation smooth scroll entre les sections

### Accessibilité
- Navigation au clavier complète
- Labels ARIA pour les lecteurs d'écran
- Support du mode reduced-motion
- Contrastes WCAG AA conformes
- Focus states visibles

### Performance
- Composants React optimisés
- Animations GPU-accelerated
- Code splitting automatique via Vite
- Chargement rapide et optimisé

## Installation

### Prérequis

- Node.js (version 18 ou supérieure)
- npm

### Étapes d'installation

1. Cloner le repository
```bash
git clone https://github.com/Enoxboo/portfolio.git
cd portfolio
```

2. Installer les dépendances
```bash
npm install
```

3. Lancer le serveur de développement
```bash
npm run dev
```

4. Ouvrir le navigateur à l'adresse indiquée
```
http://localhost:5173/
```

## Scripts disponibles

```bash
npm run dev      # Lancer le serveur de développement
npm run build    # Créer une version de production
npm run preview  # Prévisualiser la version de production
npm run lint     # Vérifier la qualité du code
```
